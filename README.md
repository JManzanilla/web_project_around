# Around The U.S. 📸

Un proyecto interactivo que permite a los usuarios compartir fotografías de sus viajes, dar "me gusta" a sus favoritas y gestionar su perfil personal. Este proyecto se centra en la creación de una interfaz dinámica y responsiva utilizando JavaScript puro.

## 🚀 Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura semántica y diseño avanzado con Flexbox y Grid.
* **JavaScript (ES6+):** Manipulación del DOM, manejo de eventos y validación de formularios.
* **Metodología BEM:** Arquitectura de CSS organizada y escalable.
* **Diseño Responsivo:** Adaptabilidad total para dispositivos móviles, tablets y escritorio.

## ✨ Características Principales

* **Gestión de Perfil:** Edición dinámica del nombre y la descripción del usuario.
* **Galería Interactiva:** Posibilidad de añadir y eliminar tarjetas de lugares personalizados.
* **Visualización de Imágenes:** Visualización a pantalla completa de las fotografías mediante ventanas emergentes (modales).
* **Validación en Tiempo Real:** Los formularios verifican que los datos sean correctos antes de permitir el envío, mejorando la experiencia del usuario.

## 🛠️ Desafíos de Ingeniería Resueltos

* **Cierre de Modales:** Implementé lógica para cerrar ventanas emergentes al presionar la tecla `Esc` o hacer clic fuera del contenido, mejorando la usabilidad.
* **Optimización de Carga:** Estructuración eficiente de archivos CSS y JS para garantizar un renderizado rápido.
* **Interactividad Fluida:** Uso intensivo de plantillas de HTML (`<template>`) para generar contenido dinámico sin recargar la página.
